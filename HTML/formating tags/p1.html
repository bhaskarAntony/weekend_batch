Formating tags

The formating elements is used to write the special text in out webpages

some formating tags

1) b - bold text
2) strong - important text
3) i - italic text
4) em - emphasized text
5) mark - marked or highlighting the text
6) small - Smaller text
7) del - deleted text
8) ins - inserted  text
9) sub - Subscripted text
10) sup - Superscript text
